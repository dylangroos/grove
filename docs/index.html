Run coding agents in parallel.

Each agent gets its own branch and worktree. Snap in to check on it, snap out, let it work.

$ curl -fsSL grove.wtf/install.sh | bash
requires bash 4+ · git · dtach
⌄
One command, one agent
gr feat/auth "add JWT auth" creates a branch, a worktree, and starts your agent with that prompt. Background by default.
Snap in, snap out
gr attach feat/auth drops you into the agent session. Ctrl+\ detaches. The agent keeps working while you're gone.
See everything at a glance
gr status shows every worktree, which agent is running, and what it last committed. One screen, all your agents.
Git commands you know
checkout, status, log, diff, rm. Same words, same muscle memory. No new mental model to learn.

How it works

# Start an agent on a new branch (runs in background) $ gr checkout feat/auth "add JWT middleware" Creating worktree: feat/auth Branch 'feat/auth' not found, creating from current HEAD ✓ Worktree created: .worktrees/feat-auth ✓ Agent started (background): claude Prompt: "add JWT middleware" Attach: gr attach feat/auth # Start another one $ gr fix/login "fix the redirect loop on login" ✓ Worktree created: .worktrees/fix-login ✓ Agent started (background): claude # See what's happening $ gr status BRANCH AGENT STATUS LAST ACTIVITY feat/auth claude running 3m ago · "add JWT middleware" fix/login claude waiting "fix redirect loop" # Snap into an agent $ gr attach feat/auth (you're now inside the agent session — Ctrl+\ to detach) # See what the agent changed $ gr log feat/auth a1b2c3d add JWT middleware and auth routes f4e5d6c add token validation helper # Done — kill agent, remove worktree $ gr rm feat/auth Agent killed ✓ Removed: feat-auth

Configuration

# Run gr init to set up both files interactively $ gr init Global config (~/.grove) Agent (coding agent to use): [claude] > ✓ Created ~/.grove Detected: node (pnpm) Setup (runs once per worktree, empty line to skip): [pnpm install] > ✓ Created Grovefile
# ~/.grove — pick your agent (global, all repos) GROVE_AGENT="claude" # GROVE_AGENT_FLAGS="--dangerously-skip-permissions" # override default flags # Grovefile — per-repo setup (optional) GROVE_PROJECT="myapp" setup() { npm install }
