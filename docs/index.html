Parallel dev environments

Manage multiple isolated development environments using git worktrees and tmux. Configured per-repo via a Grovefile.

$ curl -fsSL grove.wtf/install.sh | bash
Isolated instances
Each instance gets its own git worktree, tmux session, and isolated ports. Work on multiple branches simultaneously without conflicts.
Grovefile config
Define your project with a single Grovefile — port schemes, tmux windows, shared services, health checks. Like a Makefile for dev environments.
Shared services
Idempotently start databases, caches, and other shared services with grove_ensure_service. Start once, share across all instances.
One command
grove feat/auth — creates the worktree, installs deps, starts services, and waits for healthy. grove rm feat/auth to tear it all down.
