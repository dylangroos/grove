Branch = environment
grove feat/auth creates a worktree and tmux session. Branch name in, isolated environment out. No instance numbers, no slot management.
Zero config
Works on any git repo with no setup. Add a Grovefile for setup hooks and tmux windows when you want them.
Idempotent
Run grove feat/auth ten times — it creates what's missing, skips what exists, and prints status. Safe for scripts, CI, and agents.
Six commands
grove, attach, ls, rm, help, version. One mental model. Nothing to memorize, nothing to configure.

Configure with a Grovefile

# Grovefile — drop in your repo root grove_setup() { npm install } grove_windows() { grove_window "server" "npm run dev" grove_window "ui" "cd frontend && npm run dev" }
